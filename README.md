# 🎯 Jogo do Número Secreto

Aplicação web interativa onde o usuário tenta adivinhar um número secreto gerado aleatoriamente pelo sistema.

## 🚀 Tecnologias utilizadas

* JavaScript
* React
* HTML
* CSS

## 💡 Funcionalidades

* Geração de número aleatório
* Validação de tentativas do usuário
* Feedback em tempo real (maior ou menor)
* Contador de tentativas

## 🎮 Como funciona

O sistema gera um número secreto e o usuário deve tentar adivinhar.
A cada tentativa, o jogo informa se o número digitado é maior ou menor que o número correto, até que o usuário acerte.
